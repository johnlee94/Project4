Welcome!

This is a starter app to demonstrate Authentication with the MEAN stack!

Since all controllers will be loaded inside of MainController. You'll always be able to access the current user with: 'currentUser' (in curlies) from a template.

From a controller file, you'll be able to get the current user with with $scope.$parent.currentUser. To see an example, checkout 

/client/js/controllers/tweetsCtrl.js
